sun_rise_set.location_eph ( name) 
Returns solar data for a given location.

This function retrieves (or calculates and caches) information about a location such as:
latitude, longitude, timezone, city name, country, sunrise and sunset times,
and duration of day/night. Data is saved to 'loactions.json' for reuse.

:param name: Name of the location (city, address, or 'local').
:type name: str

:return: On success, a list with:
    [0] sunrise datetime,
    [1] sunset datetime,
    [2] day length in hours,
    [3] night length in hours,
    [4] day segment length (Wadokei) in hours,
    [5] night segment length (Wadokei) in hours,
    [6] sunrise in decimal hours,
    [7] sunset in decimal hours,
    [8] number of days since last December 7.

    On failure, returns error code:
    2 - Location not found,
    3 - Connection error to map service,
    4 - Unknown error during geolocation,
    5 - Insufficient location data,
    6 - No sunrise/sunset in this location.

:rtype: list | int
