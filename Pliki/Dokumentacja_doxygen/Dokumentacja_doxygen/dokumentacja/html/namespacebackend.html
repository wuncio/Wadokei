◆ change_time_zone()

backend.change_time_zone ( str location) 
Changes the time zone and updates all related global values such as
day length, night length, sunrise, and sunset times.

:param location: A string specifying the location ("local" or city name).
:type location: str

◆ check_date()

backend.check_date ( ) 
Flask route to calculate the current Chinese zodiac sign and the angle
for that segment on the Wadokei-style zodiac wheel.

:return: JSON object with days remaining in sign and angle value.
:rtype: Response

◆ get_local_coors()

backend.get_local_coors ( ) 
Retrieves the user's approximate location based on their IP address using ipapi.
Falls back to a default location if network access fails.

:return: Coordinates and sun-related data from location_eph().
:rtype: tuple

◆ get_timezone_info()

backend.get_timezone_info ( ) 
Flask route to return current local time data and related astronomical values
used for the Wadokei (Japanese clock) simulation.

:return: JSON object with time and sun position data.
:rtype: Response

backend.set_timezone ( )
